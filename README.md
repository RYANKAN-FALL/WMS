# WMS

Warehouse Management System built with Next.js (App Router), Prisma, and NextAuth.

## Development
1. Install deps: `npm install`
2. Seed demo data: `npm run db:seed`
3. Run: `npm run dev` then open http://localhost:3000

Demo logins: `admin/password`, `staff/password`, `supervisor/password`. If 2FA diaktifkan, gunakan OTP `246810` (atau `NEXT_PUBLIC_DEMO_OTP`).

## API Ringkas
Lihat `api.md` untuk daftar endpoint (auth, settings, orders, produk, laporan CSV, webhook tests).
